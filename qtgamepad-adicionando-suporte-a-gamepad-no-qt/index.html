QtGamepad: Adicionando suporte a Gamepad no Qt

Alexandre Vicenzi em Notícias
14/05/2015 · 1 minuto de leitura

Atenção

Um dos mais novos módulos adicionados ao Qt Labs é o QtGamepad, um módulo inspirado na API Gamepad do HTML5. Apesar de inspirado no Gamepad HTML, ele segue o estilo de desenvolvimento do Qt.

O QtGamepad oferece APIs C++ e Qt Quick a também existe a possibilidade de se criar novos backends graças a arquitetura plug-in. Com ele é possível obter os eventos do dispositivo, como por exemplo, pressionar um botão ou obter estados.

Atualmente os backends para suporte dos gamepads são o X Input no Windows, evdev no Linux e SDL2 para qualquer plataforma suportada pelo SDL2. O backend para suporte a OS X / iOS ainda está em desenvolvimento.

Confira mais informações no anúncio feito na lista de e-mails do projeto.
